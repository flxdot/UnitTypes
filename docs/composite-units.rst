Composite Units
===============

For this package Composite Units are defined as units which are able to be displayed by a multiplication or division
of other units.

For Example meters, kilograms and seconds are Base Units. But the combination of all meters / seconds creates a new unit
Newton:

.. code-block::

  N = kg * m / s\ :sup:`2`
